# Node事件循环
![](../../images/frontend/event_loop.png)

```javascript
// test.js
setTimeout(() => console.log(1));
setImmediate(() => console.log(2));
process.nextTick(() => console.log(3));
Promise.resolve().then(() => console.log(4));
(() => console.log(5))();

// 5, 3, 4, 1, 2
```
