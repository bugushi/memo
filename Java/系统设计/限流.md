# 限流的场景
缓存、降级 和 限流 被称为高并发、分布式系统的三驾马车
- 网关层限流（nginx，gateway）
    > 如果网关是集群，也得依赖中间件限流
- 中间件限流（redis, sentinel)

# 限流的常用方案
- 固定窗口
- 滑动窗口
- 漏桶
- 令牌桶
